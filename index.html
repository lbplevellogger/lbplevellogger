About

LBP Level Logger is a small system tray app designed to log LittleBigPlanet level statistics over time from lbp.me for graphing/visualisation. It supports LBP1, LBP2, LBP3, LBP Vita and LBP Karting levels. It is written in Free Pascal/Lazarus. Windows and Linux versions are available; other platforms may also build.

Change Log

  • Version 1.31 (17 Dec 2012)
    • Update to handle lbp.me website changes.
  • Version 1.3 (16 July 2012)
    • Added Windows Installer.
    • Added Linux 64 bit build.
    • Removed 'Boos' column from main display and added 'Lists' column.
    • New graphing options added:
      • It is now possible to graph a level's statistics over the past x days rather than for all time.
      • It is now possible to graph the rate of fields rather than their cumulative totals. For instance, you can now see how your plays per hour/day/week vary over time.
    • Added Preferences page.
    • Added option to disable notification area use (this is now the default on non-Windows).
    • Added option to run automatically at logon.
    • Added option to use LBP1/LBP2 specific icons instead of generic level icon.
    • Level order can now be rearranged by right-clicking a level in the level list.
  • Version 1.2 (16 Nov 2011)
    • Redesigned UI
    • 'Completed/Unique Plays Ratio' field replaced with 'Completed/Total Plays Ratio' field, and 'Total Plays/Unique Plays Ratio' field added. CSV files will be automatically updated.
    • A sensible CSV file location will now be automatically suggested when adding a new level.
  • Version 1.1 (22 Oct 2011)
    • Added CliChart integration.
  • Version 1.0 (15 Oct 2011)
    • Initial release.
